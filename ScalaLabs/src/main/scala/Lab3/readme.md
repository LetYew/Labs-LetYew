## Что сделано
В итоге получен виджет, репрезентирующий систему для заказа напитков. 

Позволяет заказать напиток, добавить к нему Доп. и т.д.


## Структура проекта


- **Lab3Main**
  - 

      Основной файл:
        Создает объект Меню 
        Генерирует: 
            - начальные примеры напитков
            - начальный пример производителя
            - вызывает GUI

- **MenuClasses**
  - 

      Для хранения классов организации меню 
    - *Drink*
        
      Содержит case class Drink, Addition и Producer, базовые компоненты меню
    - *myMenu*

      Содержит Map[...] репрезентирующий меню напитков

- DataStorage
  - 

      Директория для файловой системы:
        заглушка, но тут бы лежали списки всякого. 


- PackageGUI
  - 

      Директория для GUI:
        все что касается GUI. 

    - *GUI*
  
        Основной класс GUI, содержит виджет с основным функционалом
    - *Action*

        Класс действий, привязываемых к кнопке
    - *ButtonManager*

        Связывает кнопку, Action и LazyList[promise[Action]]. На этом строятся кнопки в GUI
    - *ButtonContainer*

        Содержит ButtonManager-ы. не задействован, но пусть лучше будет
    - *DialogBuilder*

        Конструкторы диалоговых окон для удобства
